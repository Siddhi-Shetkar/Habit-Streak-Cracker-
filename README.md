# Habit Tracker

## Overview
A simple habit tracking application that helps users build daily habits by tracking progress and maintaining streaks. The app displays a 30-day calendar and stores all data using localStorage.

## Features
- Add and manage habits
- Mark daily progress
- Track habit streaks
- View a 30-day calendar
- Persistent data using localStorage

## Tech Stack
- HTML
- CSS
- JavaScript
- LocalStorage API
